# Blueprint Técnico - Loja Moda

## Visão Geral
Plataforma de e-commerce para venda de roupas, relógios e acessórios.

## Objetivos
- Vender roupas (masculino, feminino, infantil)
- Vender relógios (luxo, esportivo, casual)
- Vender acessórios (bolsas, cintos, bijuterias, óculos)

## Canais de Atendimento
- WhatsApp / Telegram (via OpenClaw Gateway)
- Site / E-commerce
- Instagram DM

## Agentes
- **Atendente**: Recebe pedidos, tira dúvidas, envia catálogo
- **Vendas**: Fecha vendas, aplica cupons, processa pagamentos
- **Estoque**: Verifica disponibilidade, atualiza inventário
- **Pós-venda**: Rastreamento, trocas, avaliações

## Stack Tecnológico
- OpenClaw Gateway (porta 9091)
- N8N para automação de fluxos
- API de pagamento (Stripe / Mercado Pago)
- Banco de dados: PostgreSQL

## Integrações
- Catálogo de produtos
- Sistema de estoque
- Gateway de pagamento
- Transportadora / Logística
